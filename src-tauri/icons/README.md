# Tauri Icons

Place the following icon files here for the application bundle:

- `32x32.png` — 32×32 PNG icon
- `128x128.png` — 128×128 PNG icon
- `128x128@2x.png` — 256×256 PNG icon (2× retina)
- `icon.icns` — macOS icon bundle
- `icon.ico` — Windows icon

Generate these from a source SVG/PNG using:
```bash
npx @tauri-apps/cli icon path/to/source-icon.png
```
